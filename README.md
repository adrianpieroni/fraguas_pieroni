# Fraguas fraguas_pieroni - Catálogo Web

Sitio web desarrollado para un microemprendimiento real dedicado a la fabricación de fraguas artesanales de alta eficiencia para herrería, cuchillería y artesanías.

## 🚀 Características del Proyecto
* **Catálogo Técnico:** Presentación detallada de los modelos de fraguas, especificaciones de temperatura y aislamiento.
* **Diseño Adaptativo (Responsive):** Optimizado para que los clientes puedan navegar cómodamente desde el celular o la computadora.
* **Canal de Ventas Directo:** Integración con botones de contacto rápido hacia WhatsApp e Instagram para asesoramiento técnico y cierre de pedidos.

## 🛠️ Tecnologías Utilizadas
* HTML5 (Estructura semántica)
* CSS3 (Diseño, maquetación y estilos temáticos)
* JavaScript (Interacciones)
* Alojamiento: GitHub Pages
